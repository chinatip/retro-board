# retro-board
### Features :bulb:
- Basic kanban board
    - New column
    - New card
    - Movable cards
    - Merge cards
    - Vote
- Useful features during retrospective meeting
    - Blur cards
    - Hide votes
    - Mark to action items
    - Who's DONE with writing/voting
- Store board
- Store board per user
- User can have more than 1 board
- Export board as image
